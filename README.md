# Formatly - Professional Resume Builder

Formatly is a modern, single-page application (SPA) designed to help users create ATS-friendly resumes and documents instantly.

## Features

-   **Resume Builder**: Step-by-step wizard to create professional resumes.
-   **ATS Optimization**: Real-time ATS score analysis and feedback.
-   **Live Preview**: Instant preview of your document as you type.
-   **DOCX Export**: Download your resume in Microsoft Word format.
-   **AI Assistance**: (Mock) AI tools for summary generation and job matching.

## Tech Stack

-   **Frontend**: Vanilla JavaScript (ES Modules), HTML5, CSS3
-   **Build Tool**: Vite
-   **Libraries**: `docx`, `file-saver`

## Getting Started

1.  Clone the repository.
2.  Run `npm install` to install dependencies.
3.  Run `npm run dev` to start the development server.
4.  Build for production with `npm run build`.
